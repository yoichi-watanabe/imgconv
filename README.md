## コマンド実行方法

### imgconv [ options ] [< args >]

コマンド実行例 : imgconv -s png -d gif -t ./

- -s 変換前画像形式(デフォルト:jpg)
- -d 変換後画像形式(デフォルト:png)
- -t 変換対象ディレクトリ(デフォルト:./)
